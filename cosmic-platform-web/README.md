# cosmic-platform-web

Cosmic Platform - Web Repository

## 🌌 Overview

Web layer of the Cosmic Platform - a federated social platform treating content as living entities in a cosmic ecosystem.

## 🚀 Quick Start

```bash
# Install dependencies
pnpm install

# Development
pnpm dev
```

## 📁 Project Structure

```
cosmic-platform-web/
├── src/           # Source code
├── tests/         # Test files  
├── docs/          # Documentation
└── README.md      # This file
```

## 🛠️ Technology Stack


- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + shadcn/ui
- **State Management**: React Query + Zustand
- **Animation**: Framer Motion
- **PWA**: next-pwa
- **Testing**: Jest + Testing Library

## 🎨 Component Development

```bash
# Start Storybook
pnpm storybook

# Run tests
pnpm test
```

## 📱 Features

- 🌟 Star System management
- 🪐 Planet navigation (Mercury, Venus, Mars, Jupiter, Earth)
- 🌉 Bridge creation and management
- 🌌 Galaxy communities

- 📱 Progressive Web App
